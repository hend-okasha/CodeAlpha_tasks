# CodeAlpha Internship Projects

This repository contains **three Java projects** developed during my internship at CodeAlpha. Each project demonstrates **Java programming, OOP, and file handling**.

## Projects

### 1. Student Grade Tracker
- Manage student grades.
- Calculate average, highest, and lowest scores.
- Display summary reports.
- **Technologies:** Java, ArrayList, OOP.

### 2. Stock Trading Platform
- Simulate stock trading with live price changes.
- Buy/sell stocks and track portfolio.
- Save portfolio data using **File I/O**.
- **Technologies:** Java, OOP, File I/O, Multithreading.

### 3. Hotel Reservation System
- Search, book, and manage hotel rooms.
- Room categories: Standard, Deluxe, Suite.
- Payment simulation and booking details.
- Store bookings using **File I/O**.
- **Technologies:** Java, OOP, File I/O.
